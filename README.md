# Curriculum Builder System

A comprehensive, GUI-based curriculum management system built with Python, designed to handle academic data structures including departments, courses, and instructors.

## Technical Highlights
* **Relational Database Management:** Integrated SQLite for robust and persistent data storage, executing efficient CRUD operations.
* **Object-Oriented Programming (OOP):** Designed a highly modular architecture by strictly separating data models (`models.py`), database interactions (`database.py`), and the user interface (`gui.py`).
* **Data Processing:** Engineered to seamlessly process and link relational academic data, ensuring data integrity across different modules.

## Technologies
* Python
* SQLite
